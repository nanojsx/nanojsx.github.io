Lifecycle

If you extend from Component you will have willMount(), render() and didMount().

If you use SSR, you should only manipulate the DOM inside didMount(), since didMount() will not get executed on the server-side.

Update/Re-render Component

Nano-JSX does never update the component automatically. You have to call update(). On every component you call update(), the root element needs to be a DOM element or a Fragment with at least one DOM node as direct child. Also, there is no this.state, you can simply use any name you want ("data" in the example below).
